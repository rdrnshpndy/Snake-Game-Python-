# Snake Eater — Classic Snake Game

A recreation of the classic Snake game built in Python using the Pygame library.

## Overview

This project implements the well-known Snake game, where the player navigates a growing snake to eat food while avoiding collisions with the walls or itself. The codebase is fully commented and structured for easy customization.

## Features

- Classic snake gameplay mechanics
- Adjustable difficulty levels via a configurable variable
- Clean, well-commented code for easy modification of gameplay or visuals

## Installation

Make sure Python 3 is installed, then install Pygame:

```bash
pip3 install pygame
```

## Running the Application

Download the source code and run it like any other Python script:

```bash
python3 "Snake Game.py"
```

## Customization

The `difficulty` variable can be adjusted to change the game's speed and challenge level. Refer to the in-code comments for the available options. The rest of the code is self-explanatory and can be tweaked to change visuals or play style.

## Prerequisites

- [Python 3](https://www.python.org)
- [Pygame](https://www.pygame.org/wiki/GettingStarted) — an open-source Python library for building multimedia applications
